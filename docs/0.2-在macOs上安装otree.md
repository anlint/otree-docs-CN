### macos otree

- 下载安装最新版 python
- 进入

```
/Applications/Python\ 3.6/Install\ Certificates.command
```

- 安装otree

```
pip3 install -U otree
```

- 运行otree

```
otree startproject oTree
```

- 进入路径

```
cd oTree
```

- 运行服务

```
otree devserver
```

- 打开浏览器，访问 http://localhost:8000，运行实验
