### windows

- 下载并安装 python 3.6，并且添加到环境变量中，多版本时注意修改环境变量的 path，则可以多版本共存
- 安装 otree

```
pip3 install -U otree
```

- 开启otree

```
otree startproject oTree
```

```
cd oTree
```

- 开启服务器

```
otree devserver
```

**打开浏览器， http://localhost:8000/， 即可访问实验**

notes: 更好的python代码编辑体验，请下载pycharm
